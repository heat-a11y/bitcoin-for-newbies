# Bitcoin for Newbies 🧡

An interactive, beginner-friendly introduction to Bitcoin — no jargon, no gatekeeping. Built for anyone who's ever asked "what is Bitcoin, actually?"

## 📱 Android App

Also available as an Android app: **Bitcoin Widget** — a simple WebView wrapper for quick access.

### Build from source

```bash
./gradlew assembleDebug
```

APK: `app/build/outputs/apk/debug/app-debug.apk`

## 🧠 What you'll learn

- Why Bitcoin was created (the Cypherpunk dream)
- How mining works (in simple terms)
- Wallets, keys, and self-custody
- Common myths debunked
- How to buy your first sats

Open `index.html` in a browser to start learning.
